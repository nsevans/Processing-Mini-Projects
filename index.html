P5.js Mini Projects

- By Nicholas Evans

Here are a repository of projects that I've made using the p5.js library. Project ideas come from either Coding Train Challenges, Numberphile videos, other YouTube videos, random places on the internet, or ideas that I thought would be fun to create.

I like to add interactions to my projects, so a lot of the projects below will options at the bottom to be able to change different variables and see the effects.

  • Raycasting
  • Noise Map
  • 2D Flocking Simulation (Boid)
  • Conway's Game of Life
  • Chaos Game - Sierpinski Triangle
  • Kinematics
  • Fireworks
  • Square Filler
  • Prime Counter
  • Worley Noise
